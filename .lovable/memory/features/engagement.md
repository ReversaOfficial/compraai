---
name: Engagement Features
description: Favorites, reviews, notifications, popup ads, recent searches system
type: feature
---
Tables: favorites, store_favorites, product_reviews, store_reviews, notifications, notification_reads, recent_searches, popup_ads
Products/stores have avg_rating + review_count columns.
Only users who purchased can review.
Admin can moderate reviews (toggle visibility), send notifications, manage popup ads.
Popup ads support frequency: always, once_per_session, once_per_day.
Search has autocomplete with products/stores/categories suggestions.
Account page has tabs: Pedidos, Favoritos, Avaliações, Notificações, Dados, Endereço.
